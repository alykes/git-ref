General git help  
`git help`

List all commands  
`git help -a`

List common git guides  
`git help -g`

More help information on a command  
```git help <command>```
```
eg 
git help rebase
git help reset
```  

**Note:** All the commands can instead be appended with `--help`  
That is git <command> --
help

```
git glossary --help
git rebase --help
```

**Note:** Use the `-p` or `--paginate` switch to pasue the output at each page.

Git configuration  
```git config help```

Git glossary  
```git help glossary```
